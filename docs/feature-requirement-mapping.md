## Group Case Mapping

This mapping demonstrates how every feature of **WatchMate: AI Movie Recommendation System** is connected to the system requirements, user stories, wireframes, and final prototype. Each screen has a specific purpose and supports the project's MVP by helping users discover movies through AI-powered recommendations, search functionality, detailed movie information, watchlist management, and chatbot conversation history.
