# IP Summary

This project contains both original work and third-party resources.

## Original Assets

- HTML, CSS, and JavaScript source code created by the WatchMate Team.
- WatchMate user interface and prototype design.
- AI Movie Recommendation System documentation.
- System architecture diagrams and workflow diagrams.

## Third-Party Assets

The project uses the following third-party resources:

- **Tailwind CSS** for responsive interface styling.
- **Plus Jakarta Sans** from Google Fonts for typography.
- **Google Material Symbols** for icons.
- **TMDB (The Movie Database) API** for movie information and metadata.
- **OpenAI API** (or another AI service) for chatbot-powered movie recommendations.
- **Mermaid.js** for technical diagrams.

These resources are used according to their respective licenses and remain the property of their original owners.

## External APIs and Datasets

The prototype uses external services to improve the user experience:

- **TMDB API** for movie titles, genres, ratings, posters, trailers, and descriptions.
- **OpenAI API** (or another AI service) for generating personalized movie recommendations.
- Sample user data is used during prototype demonstrations. No real personal data is stored or shared.
