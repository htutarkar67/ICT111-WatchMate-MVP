# Experiment Script

## 1. Opening Script

Thank you for participating in our usability test. This is **not a test of you**—we are testing whether our WatchMate prototype is easy to understand and use.

As you complete each task, please **think aloud** by telling us what you are thinking, what you expect to happen, and anything that seems confusing or difficult.

---

## 2. Test Scenario

### Scenario: Movie Viewer

Imagine you want to watch a movie but are unsure what to choose. You decide to use **WatchMate**, an AI-powered movie recommendation platform, to help you find a movie based on your preferences.

Use the prototype as you normally would.

---

## 3. Tasks for Tester

| Task No. | Task Instruction | Related Requirement | Expected Result |
|----------|------------------|---------------------|-----------------|
| **T1** | Open the homepage and explain what WatchMate is used for. | FR-01, FR-02 | User understands the purpose of the system. |
| **T2** | Start a conversation with the AI chatbot and describe the type of movie you want to watch. | FR-03, FR-04 | User successfully interacts with the chatbot. |
| **T3** | View the personalized movie recommendations generated by the chatbot. | FR-05 | User understands and reviews the recommended movies. |
| **T4** | Open a movie's detail page to view additional information. | FR-06 | User can easily access and understand the movie details. |
| **T5** | Browse the trending movies or access previous chatbot conversations. | FR-07, FR-08 | User successfully navigates the additional features without assistance. |

---

## 4. Observation Notes

Record any observations while the tester performs the tasks.

### General Observations Table

| Observation | Task No. | Notes |
|--------------|----------|-------|
| Hesitation or long pauses | T2 | Tester hesitated when deciding what to type to the AI chatbot. |
| Clicked the wrong button/page | T5 | Tester initially selected the wrong menu before finding Trending Movies. |
| Asked questions about the interface | T3 | Tester asked how the AI generated the recommendations. |
| Confused by labels or navigation | T5 | Tester was unsure about the difference between Chat History and Trending Movies. |
| Made mistakes during task completion | T2 | Tester entered a very short prompt and expected more specific recommendations. |
| Positive comments | T1, T3 | Homepage was easy to understand and movie recommendations looked useful. |
| Suggestions for improvement | T2, T3, T5 | Add example chatbot prompts, improve recommendation explanations, and make navigation more intuitive. |

### Task-Specific Notes

- **T1 (Homepage):** Most testers quickly understood the purpose of WatchMate from the homepage.
- **T2 (AI Chatbot):** Some testers wanted example prompts to help them start the conversation.
- **T3 (Movie Recommendations):** Testers liked the personalized recommendations but wanted a brief explanation of why each movie was recommended.
- **T4 (Movie Details):** Users found the movie information clear and appreciated the clean layout.
- **T5 (Trending Movies / Chat History):** Some testers initially had difficulty distinguishing between the Trending Movies and Chat History pages.

---

## 5. Closing Questions

- **What was easy to understand?**  
  The homepage, AI chatbot interface, and movie recommendation results.

- **What was confusing?**  
  Some chatbot prompts and the navigation between Trending Movies and Chat History.

- **Which feature was most useful?**  
  The AI-powered personalized movie recommendations.

- **What should be improved before the final prototype?**  
  Add suggested chatbot prompts, provide explanations for recommendations, improve navigation labels, and enhance the chatbot conversation flow.
