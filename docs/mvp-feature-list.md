# Lab 04 - MVP Feature List

## MVP Decision Rule

The Minimum Viable Product (MVP) focuses on helping users discover movies efficiently through an AI-powered chatbot. Users can browse trending movies, search for titles, receive personalized recommendations, save chat history after logging in, manage a watchlist, and view detailed movie information. Administrators can monitor chatbot conversations through the admin dashboard.

---

## MoSCoW Prioritization

| Feature ID | Feature Name                    | Description                                                                                                       | Priority        |
| ---------- | ------------------------------- | ----------------------------------------------------------------------------------------------------------------- | --------------- |
| **F01**    | AI Movie Recommendation Chatbot | Users can communicate with the AI assistant using natural language to receive personalized movie recommendations. | **Must-Have**   |
| **F02**    | Movie Search                    | Users can search movies and TV shows by title.                                                                    | **Must-Have**   |
| **F03**    | Browse Trending Movies          | Users can browse trending movies and TV shows from the homepage.                                                  | **Must-Have**   |
| **F04**    | Movie Detail View               | Users can view movie details including synopsis, genre, release date, rating, poster, and trailer.                | **Must-Have**   |
| **F05**    | User Login                      | Users can create an account and log in to access personalized features.                                           | **Must-Have**   |
| **F06**    | AI Chat History                 | Logged-in users can view and continue previous AI conversations.                                                  | **Must-Have**   |
| **F07**    | Watchlist                       | Users can save movies they plan to watch later.                                                                   | **Should-Have** |
| **F08**    | Watch Trailer                   | Users can watch official movie trailers directly from the application.                                            | **Should-Have** |
| **F09**    | Admin Chat History Dashboard    | Administrators can view user chatbot conversations for monitoring and analysis.                                   | **Could-Have**  |
| **F10**    | Personalized AI Learning        | The AI improves recommendations by learning from users' long-term preferences and watch history.                  | **Not in MVP**  |

---

## Must-Have (Core MVP)

These features demonstrate the core value of WatchMate.

* **F01 (AI Movie Recommendation Chatbot):** Enables users to receive personalized recommendations through natural conversation.
* **F02 (Movie Search):** Allows users to quickly search for movies and TV shows.
* **F03 (Browse Trending Movies):** Helps users discover popular content without searching.
* **F04 (Movie Detail View):** Displays comprehensive movie information to support viewing decisions.
* **F05 (User Login):** Allows users to access personalized features and save their activity.
* **F06 (AI Chat History):** Saves previous chatbot conversations so users can revisit earlier recommendations.

---

## Should-Have (High Priority)

These features enhance the user experience.

* **F07 (Watchlist):** Enables users to save movies they want to watch in the future.
* **F08 (Watch Trailer):** Allows users to preview movies before deciding to watch them.

---

## Could-Have (Future Enhancements)

These features provide additional management capabilities.

* **F09 (Admin Chat History Dashboard):** Enables administrators to monitor chatbot conversations and identify common user preferences.

---

## Not in MVP (Out of Scope)

These features require more advanced AI development and are beyond the scope of the first prototype.

* **F10 (Personalized AI Learning):** Automatically adapts recommendations based on long-term user behavior and viewing patterns.
