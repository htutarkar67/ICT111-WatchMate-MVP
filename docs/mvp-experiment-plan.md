# MVP Experiment Plan

## 1. Group and Project Information
- Group name: WatchMate
- Project title: WatchMate - AI Movie Recommendation System
- Repository link: https://github.com/htutarkar67/ICT111-WatchMate-MVP
- Main target user: Movie viewers, students, and busy individuals
- Prototype platform: Interactive Clickable Web Prototype

## 2. Experiment Objective
We want to test whether users can successfully interact with the AI chatbot, describe their movie preferences, receive personalized movie recommendations, browse movie details, and navigate the system without confusion.

## 3. Requirement Scope for the Experiment

| Requirement ID | Requirement Summary | Related Screen/Feature | Tested in This Experiment? |
|---|---|---|---|
| FR-01, FR-02 | Homepage / Landing Page | `index.html` | Yes |
| FR-03, FR-04 | AI Chatbot Conversation | `chat.html` | Yes |
| FR-05 | Personalized Movie Recommendations | `recommendation.html` | Yes |
| FR-06 | Movie Details | `movie-detail.html` | Yes |
| FR-07 | Trending Movies | `trending.html` | Yes |
| FR-08 | Chat History | `history.html` | Yes |

## 4. MVP Experiment Type

**Selected Experiment Type:** Interactive Clickable Prototype

**Reason for Selection:**  
We developed an interactive frontend prototype that simulates the AI-powered movie recommendation workflow. This enables us to evaluate user interactions, navigation, and overall usability before implementing the full AI recommendation system.

## 5. Test Users

| Test User Group | Number of Testers | Why They Are Relevant |
|---|---:|---|
| Students | 4 | They are the primary target users who frequently watch movies and need quick, personalized recommendations. |
| Movie Viewers | 1 | They represent users who regularly search for movies and can evaluate the usefulness of the recommendation experience. |

## 6. Experiment Procedure Summary

1. Provide testers with access to the interactive prototype.
2. Ask users to explore the homepage and understand the purpose of WatchMate.
3. Ask users to interact with the AI chatbot by describing the type of movie they want to watch.
4. Ask users to review the personalized movie recommendations and view detailed movie information.
5. Ask users to browse trending movies and access previous chatbot conversations.
6. Observe whether users can complete each task without assistance, noting any usability issues or confusion.
7. Ask users to complete a feedback form to evaluate usability, recommendation quality, and overall satisfaction.

## 7. Expected Learning

We will learn whether users can easily interact with the AI chatbot, understand the recommendation process, and navigate the prototype efficiently. Based on the feedback, our team will improve the chatbot conversation flow, user interface, navigation, and recommendation experience before developing the final system.
